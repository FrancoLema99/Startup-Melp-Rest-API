# Startup Melp - Rest API


Ejecutar python manage.py runserver para iniciar el servidor
Ejecutar en el Browser: http://127.0.0.1:8000/api/restaurants para acceder a la instancia de creación
Acceder a http://127.0.0.1:8000/api/restaurants/{id}/ para eliminar y/o modificar elementos
Acceder a http://127.0.0.1:8000/docs/ para ver la Documentación
Acceder a http://127.0.0.1:8000/admin/ e ingresar el usuario y contraseña para poder ver el Panel de Administrador
